# 缠论选股系统

基于缠论技术分析的智能选股Web应用

## 功能特点

- 📊 **缠论分析**：自动识别顶底分型、划分笔、计算中枢
- 🎯 **买卖点信号**：自动判断一买、二买、三买信号
- 📱 **自定义股票池**：支持手动输入或板块自动扫描
- ☁️ **云端部署**：支持部署到Streamlit Cloud，手机随时访问

## 使用方法

1. 访问部署后的链接
2. 在侧边栏选择"自定义股票池"或"板块自动扫描"
3. 点击"开始分析"按钮
4. 查看三买/一买信号股票
5. 支持导出CSV数据

## 技术栈

- Python
- Streamlit
- Tushare (股票数据)
- Pandas / NumPy

## 部署说明

本应用需要设置 `TUSHARE_TOKEN` 环境变量。
